Player

http://localhost:8080/player

    API Endpoint

        http://localhost:8080/

Realize uma requisição GET no Endpoint "/player" para trazer todos os jogadares cadastrados.

Realize uma requisição GET no Endpoint "/player/{id}" para trazer um específico jogadares cadastrado.

Para cadastrar um novo jogador utilize o método POST no Endpoint "/player".

Exemplo: {
"nameplayer": "Zappts"
}

Para alterar o nome de um jogador utilize o método PUT no Endpoint "/player/{id}" .

Exemplo: {
"nameplayer": "Zappts atualizado"
}

Para excluir um jogador utilize o método DELETE no Endpoint "/player/{id}" .
